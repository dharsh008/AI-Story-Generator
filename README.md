The AI Story Generator is an interactive, full-stack web application that leverages advanced Natural Language Processing to generate creative, personalized stories on demand. Built with a Python backend using the Flask web framework, the application integrates the lightweight distilgpt2 machine learning model via the Hugging Face Transformers library and PyTorch to handle complex text generation. The user-friendly frontend—designed with custom HTML, modern glassmorphism CSS aesthetics, and JavaScript—allows users to input custom story prompts, select specific genres (like Sci-Fi, Horror, or Fantasy), and choose their desired story length. When a user submits a request, the frontend seamlessly communicates with the backend REST API, processing the prompt through the AI model, and dynamically displaying the dynamically generated narrative directly onto the screen without requiring a page reload. It also features convenient tools allowing users to directly copy or download their generated story.
